<h1> 3D Space Shooter Game | Python, OpenGL (PyOpenGL), GLUT</h1>

• Developed a 3D space shooter game featuring real-time rendering, collision detection, enemy AI behavior, score
tracking, and difficulty progression.  <br>
• Implemented collision detection, enemy spawning, projectile systems, score tracking, and progressive difficulty
scaling to enhance gameplay experience. <br>
• Designed first-person and third-person camera modes, interactive controls, and game state management for a
responsive user experience. <br>
• Built custom 3D models, lighting effects, and procedurally generated space environments including stars, planets,
and animated enemy objects. <br>
• Applied object-oriented programming, mathematical computations, and event-driven architecture to manage
game logic and rendering performance. <br>
